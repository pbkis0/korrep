11. Lecke: CSS3 eszközök

A CSS3 jelentősen kibővítette a korábbi stílus megadási lehetőségeit, de a visszafele kompatibilitást megtartott. Kiterjesztette a kijelölési módokat és új formázási lehetőségeket vezetett be. Ez utóbbiból tárgyalunk néhányat.

Azt figyelembe kell venni, hogy ezeket a formátumokat a böngészők újabb verzióival lehet használni, régebbi verziókkal vagy nem működött, vagy a formátum elé egy előtagot kellett tenni:

  • Chrome és Safari esetén: -webkit- 
  • Firefox esetén: -moz-
  • Opera esetén: -o-
  • Explorer esetén: -ms-

Színátmenetes háttér

Szín megadásakor lehetőség van kettő vagy több szín közötti átmenet használatára is. Az átmenet lehet egyenes vonalú vagy sugár irányú.

Egyenes vonalú átmenetet a linear-gradient megadásával készíthetünk. Az átmenethez kettő vagy több színt kell megadni, az átmenetet a böngésző készíti el a megadott színeken keresztül:

background : linear-gradient(red, white, green);

Megadható az átmenet iránya a színek megadása előtt. Például ha balról jobbra szeretnénk az átmenetet, akkor a "to right" szöveget kell megadni, vagy egy átlós átmenethez a "to bottom right" szöveget. 

background : linear-gradient(to right, red, white, green);

Százalékosan megadható, hogy egy színbe való átmenet a terület mely részénél érjen véget

background : linear-gradient(to right, red, white 20%, green 80%);

A sugár irányú átmenet hasonlóan működik a radial-gradient megadásával, csak az átmenet az elem közepétől kifele a sarkai felé tart.

background: radial-gradient(green 20%, white 60%, red 80%);

Árnyékolás

Lehetőség van egyes elemeknek árnyékot adni. Itt külön választjuk a szöveg árnyékát és a "doboz", azaz tároló árnyékát. A doboz esetén az árnyék a doboz két oldalánál jelenik meg, a szöveg esetén viszont követi a karakterek vonalát.

Mindkét esetben az árnyékot ugyanúgy kell megadni: meg kell adni az árnyék méretét x és y irányban, az elmosódás mértékét és az árnyék színét.

doboz árnyék

box-shadow : 10px 10px 5px gray;

szöveg árnyék

text-shadow : 50px 15px 10px gray;

Transzformálás

Az elemeken különböző transzformációt lehet végrehajtani. 

A kívánt átalakítást a transform tulajdonságnak kell megadni.

eltolás egy vektorral

transform : translate(100px, 200px);

elforgatás egy szöggel

transform : rotate(90deg);

átméretezés szélesség és magasság szerint

transform : scale(2,0.5);

eltorzítás x és y irányban

transform : skew(10deg, 20deg);

3 dimenzióban forgatás x, y, vagy z tengely mentén

transform : rotateX(90deg);
transform : rotateY(90deg);
transform : rotateZ(90deg);

Átmenet

Egy elem valamilyen stílus tulajdonságát meg lehet változtatni egy átmeneten keresztül. Ha az elem valamilyen tulajdonságának az értéke változik két állapota között, akkor ez a változást végrehajthatja úgy, hogy a transition tulajdonságnak meg kell adni a tulajdonságot és, hogy mennyi idő (másodperc) alatt hajtsa végre. Egyszerre több tulajdonság is változhat átmenetesen, ha vesszővel elválasztva felsoroljuk.

div {
  background-color : red;
  transition : background-color 2s;
}
div:hover {
  background-color : yellow;
}

Az átmenetet lehet késleltetni valamennyi másodperccel

transition-delay : 2s;

Változtatható az átmenet sebessége, például lehet lassan kezdni, vagy befejezni, vagy lassan kezdeni gyorsulni és lassan befejezni. 

transition-timing-function : ease / ease-in / ease-out / ease-in-out;

Animáció

A CSS3 segítségél az elemekhez animációkat lehet készíteni pusztán stílussal, programozás nélkül. Itt valójában egy elem bizonyos tulajdonságának (akár többnek is egyszerre) értéke közötti váltást lehet egy átmenettel elvégeztetni.

Először egy általános animáció állapotait kell elkészíteni, azaz, hogy az animáció során milyen tulajdonságok milyen fázisban hogyan változzanak. Az alábbi "nyulik" nevű animációt bármely elemhez hozzá lehet később rendelni.

@keyframes nyulik {
  0% { width : 100px; background-color : red; }
  50% { width : 200px; background-color : yellow; }
  100% {width : 100px; background-color : red; }
}

Ezt az animációt hozzá kell rendelni az animálandó elemhez. Figyelni kell arra, hogy az elem animált tulajdonsága illeszkedjen az animáció kezdő és végállapotához, különben éles váltás lesz az animáció előtt vagy után.

animation-name : nyulik

Megadható, hogy az átmenet mennyi idő alatt hajtódjon végre másodpercben mérve.

animation-duration : 4s;

Az animáció alapértelmezettként az oldal betöltődésekor lejátszódik. Ezt lehet késleltetni valamennyi másodperccel

animation-delay : 5s;

Az animációt többször is le lehet játszatni, vagy akár folyamatos ismétlődést is megadható. Ilyenkor arra figyelni kell, hogy az animáció végállapota illeszkedjen a kezdőállapotához (legjobb, ha ugyanaz), különben két lejátszás között éles váltás lesz.

animation-iteration-count : 3;
animation-iteration-count : infinite;

Megadható, hogy az animáció fordított irányban játszódjon le, vagy akár oda-vissza

animation-direction : reverse;
animation-direction : alternate;

Szabályozható az animáció lejátszásának sebessége, lehet gyorsuló, lassuló, vagy lassan kezdő gyorsuló és visszalassuló a végére.

animation-timing-function : ease-in / ease-out / ease;
